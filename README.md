# Soumya Portfolio

A deployable Next.js portfolio based on Sri Soumya Narasimhadevara's resume.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Notes

- Replace `public/resume.pdf` with your actual PDF resume.
- Update the GitHub placeholder link in `app/page.tsx`.
- Deploy easily on Vercel.
